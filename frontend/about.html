SB Query is a backend + frontend about Hypixel Skyblock. It's made by KTibow#3960.

Go back to the main page

The backend uses:

  • Replit (site hoster)
  • Python (programming language)
  • Sanic (server framework)
  • GraphQL (query language)
  • Various data sources

The frontend uses:

  • GraphiQL (query previewer)
  • Tailwind CSS (CSS framework)
  • Vanilla JS/HTML (scripting/layout)

The data sources are:

  • Official Hypixel API
  • NotEnoughUpdates database (Skyblock items)
